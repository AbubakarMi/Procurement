# **App Name**: GovConnect

## Core Features:

- Complaint Summarization: Use AI to summarize user complaints for faster review.
- Interactive Project Map: Display project locations on an interactive map with detailed information.
- Tender Listings: List featured tenders with descriptions, categories, closing dates, and download links.
- News Feed: Display latest news with title, snippet, and link to full article.
- Staff Directory: Provide a staff directory with images, names, roles, and contact info.
- Complaint Tracking: Allow users to track their complaints using a tracking ID.
- Resource Downloads: Organize resources with icons, titles, descriptions, and download buttons.

## Style Guidelines:

- Primary color: Cardinal red (#E34234) to convey energy and importance, based on the original requested shade of red.
- Background color: Light cream (#F9F6F6), a desaturated near-white from the same hue family.
- Accent color: Burgundy (#972D28), darker and less saturated, complementing the primary, derived from the original dark red.
- Font: 'Inter', a grotesque sans-serif for a clean, professional look, as requested.
- Note: currently only Google Fonts are supported.
- Use simple, professional icons in cardinal red to represent different resources and sections.
- Implement a mobile-first, fully responsive layout with a sticky header and uniform spacing.
- Use subtle hover effects on buttons and cards for enhanced interactivity.
